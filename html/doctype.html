<!DOCTYPE html>
  1. Case-insensitive
  2. Must come first in HTML document
  3. A DOCTYPE is a required preamble
  4. Stops browser from switching to "quirks mode"
  5. Not a HTML tag but an instruction to the browser about what version of HTML to use
  6. Including the DOCTYPE ensures that the browser makes a best-effort attempt at following specifications
