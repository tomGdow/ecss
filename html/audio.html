  1. An audio element represents a sound or audio stream
  2. controls
  3. src
    1. The URL of the audio to embed
    2. Optional; you may instead use the source element within the audio block to specify the audio to embed
  4. preload
  5. autoload
  6. loop

Example

HTML

    <audio controls>
        <source src="../audio/car.ogg">
        <source src="../audio/car.mp3">
    </audio>

Notes

  1. .ogg plays on Firefox but not on Safari
  2. Audio files created with Mathematica
