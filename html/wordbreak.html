DONT ADD ANYTHING HERE!

  1. The word-break CSS property sets whether line breaks appear wherever the text would otherwise overflow its content box
  2. normal
    Use the default line break rule
  3. break-all
    To prevent overflow, word breaks should be inserted between any two characters
  4. keep-all
    Word breaks should not be used for Chinese/Japanese/Korean (CJK) text. Non-CJK text behavior is the same as for normal.
  5. break-word
    Has the same effect as overflow-wrap: anywhere combined with word-break: normal, regardless of the actual value of the overflow-wrap property
