  1. The translate global attribute is an enumerated attribute that is used to specify whether an element's translatable attribute values and its Text node children should be translated when the page is localized, or whether to leave them unchanged
  2. It can have the following values
    1. empty string or yes which indicates that the element should be translated when the page is localized
    2. no, which indicates that the element must not be translated
  3. translate="no"
  4. translate="yes"
  5. translate=""
