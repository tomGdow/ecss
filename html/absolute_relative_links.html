  1. absolute URL
    Points to a location defined by its absolute location on the web, including protocol and domain name
  2. relative URL
    Points to a location that is relative to the file you are linking from

Example (Absolute Links)

  1. http://www.mysite.com
  2. http://www.mysite.com/graphics/image.png
  3. http://www.mysite.com/help/articles/how-do-i-set-up-a-webpage.html

Example (Relative Links)

  1. index.html
  2. /graphics/image.png
  3. /help/articles/how-do-i-set-up-a-webpage.html
