letter-spacing

  1. The letter-spacing CSS property sets the horizontal spacing behavior between text characters
  2. This value is added to the natural spacing between characters while rendering the text
  3. Positive values of letter-spacing causes characters to spread farther apart, while negative values of letter-spacing bring characters closer together

CSS 
    .letterspacing {
        letter-spacing: 1em;
        border: 1px solid gray;
        padding: 1em;
    }
