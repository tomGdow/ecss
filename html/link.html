<link href="main.css" rel="stylesheet">
  1. The HTML External Resource Link Element or <link> specifies the relationship between the current document and an external resource
  2. Usually, links to stylesheet but may also link to icons, such as favicon
  3. Key attributes:
    1. href
    2. rel
    3. type
    4. media
    5. hreflang
    6. sizes
    7. title (!special)
  4. Both href and rel are compulsory
  5. May occur in body or head (stylesheet is 'body-OK') but not good practice to use in body

Example

<link href="print.css" rel="stylesheet" media="print">

Example

Notes

  1. href
    The destination of the link(s) is given by the href attribute, which must be present and must contain a valid non-empty URL
    The address of the resource is given by the href attribute
  2. rel
    A link element must have rel attribute
    The relationship between the document and the resource is given by the rel attribute
    Other values for rel are author, icon, stylesheet, alternate
    relis short for "relationship"
  3. type
    The type attribute gives the MIME type of the linked resource
    It is purely advisory
    The value must be a valid MIME type
  4. media
    The optimum media for the resource is given by the media attribute
    The default is "all"
    If the link is a hyperlink then the media attribute is purely advisory
    If the link is an external resource link, then the media attribute is prescriptive.
    Value must be a valid media query
  5. hreflang
    The language of the resource is given by the hreflang attribute.
  6. sizes
    Specifies icon size (sizes="114x114")
  7. title
    Special semantic meaning in link tag
    Specifies a preferred or alternative stylesheet

Mnemonic

  1. anchor-area-base-link all href
  2. Tim Benners-Lee: a is for anchor, href is for hypertext reference
