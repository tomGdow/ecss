DONT ADD ANYTHING HERE!

  1. inline-size:value and writing-mode:horizontal-tb is the same as width
  2. inline-size:value and writing-mode:vertical-lr is the same as height
  3. block-size behaves in reciprocal way

Example

inline-size & horizontal-tb (SameQ: width)

Hello World
CSS 
.inline_size1 {
    inline-size: 150px;
    writing-mode: horizontal-tb;
}

inline-size & vertical-lr (SameQ: height)

Hello World
CSS 

.inline_size2 {
    inline-size: 150px;
    writing-mode: vertical-lr;
}

block-size & horizontal-tb (SameQ: height)

Hello World

            writing-mode: horizontal-tb;
